// Skills
Skills auto-activate when your prompt matches a trigger phrase — no slash command needed. They run transparently in the background.
ai-prompt-review

Reviews prompts for PCI violations (card data), injection risks, token bloat, and Claude API best practices before sending to the model.

Trigger phrases
review this prompt check my system prompt prompt review
bun-run

Manages Bun package and script operations with lockfile conflict resolution, dependency issue diagnosis, and bun.lock repair.

Trigger phrases
bun install bun run bun test bun error
db-migrate

Runs Laravel migrations safely — checks the environment, detects destructive changes (DROP, TRUNCATE), and validates rollback path.

Trigger phrases
run migration migrate database rollback migration create migration
go-lint

Runs gofmt + golangci-lint on changed files, parses results, and auto-fixes formatting issues. Reports remaining lint errors with line numbers.

Trigger phrases
lint go go lint fix go style gofmt
go-test

Runs Go tests with the race detector enabled, parses test failures, diagnoses root cause, and offers targeted code fixes.

Trigger phrases
run go tests go test failing fix go test go test ./...
iso8583-parse

Decodes ISO 8583 hex strings into human-readable field maps, including full EMV TLV parsing for Field 55 chip data.

Trigger phrases
parse this ISO message decode this hex what does this message mean
payment-sign

Generates and verifies MD5-based payment gateway API signatures with sensitive key masking. Supports common hash ordering patterns.

Trigger phrases

vue-component

Scaffolds Vue 3 Composition API components following project conventions — forms, lists, modals, and PCI-safe payment UI patterns.

Trigger phrases
